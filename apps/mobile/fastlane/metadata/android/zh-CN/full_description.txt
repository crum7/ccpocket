AI 编程代理已经能够自主编写完整功能。开发者的工作变成了"审批、回答、审查"——这些判断不需要笔记本电脑，只需要你的手机。

CC Pocket 让你从手机上启动 Codex 和 Claude Code 会话，随时随地做出决策，从任何地方交付代码。

本应用与 Anthropic、OpenAI 无关，未获得两家公司的认可或推荐。

<b>从手机启动会话</b>
无需打开笔记本电脑，直接从手机启动编程会话。选择项目、设置权限模式，几秒钟即可开始编程。

<b>批量审批</b>
多个代理同时运行也不用担心。在一个列表中查看所有会话的待审批请求，一次性处理文件编辑、命令执行和工具调用。

<b>移动端优化 UI</b>
专为触屏操作设计的界面。回答代理问题、审批工具、发送反馈——这是真正的移动端体验，而不是桌面版的缩小版。

<b>丰富的提示</b>
支持 Markdown 格式和列表自动补全，让输入更便捷。从剪贴板或照片库附加图片，为代理提供视觉上下文。

<b>整理会话</b>
为会话命名，按项目管理。通过搜索和项目筛选快速找到历史对话。

<b>审查差异</b>
语法高亮的 diff 视图一览所有代码变更。查看文件统计、选择特定变更块并发送反馈。

<b>实时流式传输</b>
实时观看代理的思考和编码过程。内联 diff、思考块和结构化计划实时呈现。

<b>推送通知</b>
代理需要审批或完成任务时立即收到通知，不再错过任何权限请求。

<b>灵活的连接方式</b>
通过已保存的机器、二维码、mDNS 自动发现或手动输入 URL 进行连接。还支持通过 SSH 管理远程服务器。

<b>隐私优先</b>
代码和对话保存在你自己的服务器上。无遥测、无跟踪。CC Pocket 不收集任何个人数据。

<b>使用方法</b>
1. 安装 Bridge Server：npx @ccpocket/bridge@latest
2. 在应用中扫描二维码或输入 URL 连接
3. 从手机开始编程

仅需 5 分钟即可完成设置。

<b>使用要求</b>
• 一台已安装 Codex 或 Claude Code CLI 的 Mac 或 Linux 设备
• 手机与服务器之间的网络连接（推荐局域网或 Tailscale）
